# 🏏 22 Yards

22 Yards is a modern cricket website built to deliver a premium user experience for cricket fans.

## ✨ Features
- Live match scores
- Trending fixtures
- Featured players
- Rankings & stats
- Cricket news
- Responsive mobile navbar
- Smooth animations
- Dynamic JSON demo data

## 🛠 Tech Stack

- JavaScript
- JSON
